** Representing Graphs
- Graphs can be represented using *adjacency matrices*
- Adjacency matrices are rectangular arrays that use \( 1 \) to indicate edges

** Operations on Arrays
- NumPy provides many operations to use with arrays
  1. Column/row sums with ~sum~ (/not/ ~np.sum~).  This is useful for finding
     degrees of vertices.
  2. Matrix multiplication with ~@~ (such as ~A@A~).  This is useful for
     counting paths between vertices.

** BFS in Python
BFS algorithm:
1. Add the root to the queue.  While the queue is nonempty, do the following:
2. Set the current vertex equal to the first vertex in the queue, removing it
   from the queue.
3. If the current vertex is the target, then set the current vertex as the
   parent of the target and erase the queue.
4. If the current vertex is not the target, then do the following for each
   neighbor:
   a. set the parent of the neighbor to the current vertex.
   b. add the neighbor to the queue.
